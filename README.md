# Bug Bounty Checklist Workspace

A focused web pentest checklist workspace built with React, Tailwind CSS (v4), Vite, and React Router.

## Tech Stack

- React 19
- React Router
- Tailwind CSS v4 + Vite plugin
- Lucide React icons

## Features

- Highlighted master web pentest checklist
- Persistent progress using localStorage
- Category dropdowns with checklist checkboxes
- Tools, usage, techniques, references, and remarks grouped per item
- Essential tools overview
- Copy-ready install command blocks
- Responsive layout for desktop and mobile

## Run Locally

```bash
corepack pnpm install
corepack pnpm dev
```

Open http://localhost:5173

## Build and Lint

```bash
corepack pnpm lint
corepack pnpm build
```
